# 华云橙（Orange）模板设计规范

## 参考模板
- **来源**：Geeker-Admin（https://github.com/HalseySpicy/Geeker-Admin）
- **本地路径**：`E:\qingshi\muban\18.Geeker-Admin-master\Geeker-Admin-master`
- **预览**：`npm run dev` → http://localhost:8848
- **特点**：深色侧边栏+顶部导航的后台布局，Element Plus 风格组件
- **提取内容**：后台侧边栏结构、配色方案、输入框带图标前缀、方正按钮、卡片式表单

## 整体气质
现代企业级管理后台，专业严谨，像飞书/钉钉/Geeker-Admin。
侧边栏提供结构感，内容区留白充足，组件风格利落干净。

## 配色
- 主色：#F97316（橙色）
- 强调色：#EA580C（深橙）
- 侧边栏背景：#191A20（极深灰黑）
- 侧边栏文字：#A6A6A6（浅灰）
- 侧边栏激活文字：#FFFFFF
- 内容区背景：#F5F7FA（浅灰白）
- 卡片/头部背景：#FFFFFF
- 深色文字：#303133
- 常规文字：#606266
- 辅助文字：#909399
- 边框色：#DCDFE6
- 分隔线：#EBEEF5
- 成功色：#67C23A
- 警告色：#E6A23C

## 字体
- 主字体：`"HarmonyOS Sans", "PingFang SC", "Helvetica Neue", system-ui, sans-serif`
- 标题：font-weight: 600
- 正文：font-weight: 400
- 辅助文字：font-weight: 400, 略小号

## 登录页
- 窗口尺寸：900×560（宽屏左右分栏）
- 布局：**左右分栏**（Geeker-Admin 核心特征）
  - 左面板（340px）：橙色渐变背景，品牌 Logo + 名称 + 功能特性列表 + 装饰圆形
  - 右面板（560px）：纯白背景，头像 + "用户登录" + 表单
- 表单：
  - 输入框：完整边框 1px solid #DCDFE6 + 圆角 4px + 左侧图标前缀 + 高度 40px
  - 聚焦时边框变橙 + 淡橙阴影
  - 复选框：方形 + 橙色勾
- 按钮：方正圆角 4px，实心橙色 #F97316，hover 变 #EA580C
- 底部链接：带图标，三列排列（注册/充值/解绑）

## 主界面
- 窗口尺寸：1440×900，最小 1200×760
- 左侧：210px 深色侧边栏 #191A20（Logo + 导航菜单，激活项橙色左竖线+浅橙背景）
- 顶部：60px 白色头部栏（左：折叠按钮+页面标题，右：用户头像+下拉菜单）
- 内容区：#F5F7FA 浅灰背景 + 20px padding

## 与青拾绿的关键差异
1. 输入框：四边圆角10px → 四边圆角4px + 图标前缀
2. 按钮：胶囊形 → 方正4px圆角
3. 阴影：柔和大阴影 → 细微阴影
4. 标题栏：绿色渐变 → 白色底+橙色logo
5. 导航：宽侧边栏+深蓝背景 → 深灰黑侧边栏+橙色激活
6. 整体感觉：圆润柔和 → 方正专业（Geeker-Admin 风）
