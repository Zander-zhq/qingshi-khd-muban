# 华云橙（Orange）模板设计规范

## 参考模板
- **来源**：Geeker-Admin（https://github.com/HalseySpicy/Geeker-Admin）
- **本地路径**：`E:\qingshi\muban\18.Geeker-Admin-master\Geeker-Admin-master`
- **预览**：`npm run dev` → http://localhost:8848
- **特点**：左侧大图+右侧表单的登录页，深色侧边栏+顶部面包屑的后台布局，Element Plus 组件库风格
- **提取内容**：登录页布局结构、后台整体布局、配色方案、组件样式（输入框前置图标、圆形按钮等）

## 整体气质
现代商务风，干净利落，像飞书/钉钉等企业级产品。留白充足，呼吸感强。

## 配色
- 主色：#F97316（橙色）
- 强调色：#EA580C（深橙）
- 背景：#FFFFFF（纯白）
- 装饰色：#FFF7ED（暖白）、#FED7AA（浅橙）
- 深色文字：#1E293B
- 辅助文字：#94A3B8
- 成功色：#16A34A
- 错误色：#DC2626

## 字体
- 主字体：`"HarmonyOS Sans", "PingFang SC", system-ui, sans-serif`
- 标题：加粗但不粗犷，font-weight: 600
- 正文偏细：font-weight: 400

## 登录页
- 窗口尺寸：500×680
- 布局：纯白背景 + 右上角大面积橙色弧形装饰
- 上方：品牌 Logo 居中 + 欢迎语
- 表单居中：
  - 输入框：只有底边框（border-bottom），无四边框，聚焦时底线变橙色
  - 密码框：同样底边框风格 + 右侧"显示密码"文字按钮（不用眼睛图标）
  - 复选框：方形 + 橙色勾
- 按钮：方正小圆角（6px），实心橙色，无阴影，hover 微微上移
- 底部链接：灰色文字，hover 变橙

## 主界面
- 窗口尺寸：1360×860，最小 1100×700
- 导航：白色顶部栏 + 左侧垂直 tab（橙色竖线指示器）
- 内容区：纯白背景
- 卡片：细边框 1px #E2E8F0，圆角 8px，无阴影
- 用户下拉：白色浮层 + 细边框

## 与青拾绿的关键差异
1. 输入框：四边框 → 只有底边框
2. 按钮：胶囊形 → 方正
3. 阴影：柔和大阴影 → 几乎无阴影，用细边框
4. 标题栏：绿色渐变 → 白色背景+橙色文字
5. 导航：宽侧边栏 → 垂直tab+顶栏
6. 整体感觉：圆润柔和 → 干净利落
