# 科技黑（Dark）模板设计规范

## 参考模板
- **来源**：Vuestic-Admin（https://github.com/epicmaxco/vuestic-admin）
- **本地路径**：`E:\qingshi\muban\11.vuestic-admin-master\vuestic-admin-master`
- **预览**：`npm run dev` → http://localhost:3000
- **特点**：现代化扁平设计，独特的 Vuestic UI 组件库，支持暗色模式，导航布局灵活
- **提取内容**：暗色模式配色、组件样式、布局结构、登录页设计

## 整体气质
赛博朋克/开发者工具风，像 Vercel/Linear 等现代开发工具。暗色但精致，不粗糙。

## 配色
- 主色：#22D3EE（青蓝）
- 渐变：#22D3EE → #A855F7（青蓝到紫）
- 背景：#0A0F1E（极深蓝黑）
- 次背景：#131B2E（略浅）
- 卡片：rgba(255,255,255,0.03)（微透明白）
- 边框：#1E293B（暗灰）
- 文字主色：#E2E8F0
- 文字辅色：#64748B
- 成功色：#22D3EE
- 错误色：#F43F5E（玫红）

## 字体
- 英文优先：`"Inter", "SF Pro Display", system-ui, sans-serif`
- 数字/时钟：`"JetBrains Mono", "Consolas", monospace`
- 中文：`"PingFang SC", "Microsoft YaHei"`
- 整体偏细 font-weight: 300-400，标题 500-600

## 登录页
- 窗口尺寸：480×720
- 布局：全屏深色渐变背景（#0A0F1E → #131B2E）
- 顶部中心：微妙的渐变光晕（radial-gradient 青蓝色，opacity 0.15）
- 居中半透明卡片：
  - 背景 rgba(255,255,255,0.03)
  - 边框：1px solid rgba(34,211,238,0.15)（微透明青蓝）
  - 圆角 12px
- 输入框：深色填充 #0A0F1E + 1px 暗边框 #1E293B + 聚焦时边框渐变（青→紫）
- 按钮：渐变填充（青蓝→紫）+ 圆角8px + hover 柔和发光 box-shadow
- Logo 下方：一条细渐变线（青→紫→透明）作为品牌分隔

## 主界面
- 窗口尺寸：1500×900，最小 1200×760
- 背景：#0A0F1E
- 顶部：深色半透明导航（48px），底部 1px 渐变线（青→紫→透明）
- 左侧：窄侧边栏（56px），半透明背景，hover 平滑展开到 200px
- 内容区：深色背景，卡片用微透明白
- 用户下拉：深色浮层 + 渐变边框

## 与其他模板的关键差异
1. 唯一的暗色模板，整体深色背景
2. 使用渐变色（青→紫）而非单色
3. 发光效果替代阴影
4. 等宽字体用于数字展示
5. 半透明+模糊效果增加层次感
6. 侧边栏极窄，hover展开
7. 边框用渐变色而非实色
