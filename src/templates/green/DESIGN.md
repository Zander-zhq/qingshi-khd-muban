# 青拾绿（Green）模板设计规范

## 参考模板
- **来源**：原创设计，项目自研
- **特点**：青绿色系，竖向单列登录页，宽侧边栏后台布局，PrimeVue 组件库
- **说明**：这是项目的默认模板，所有新功能优先在此模板上开发和测试

## 整体气质
清新自然、亲和力强，像一个面向普通用户的工具软件。

## 配色
- 主色：#0D9488（青绿）
- 渐变：#2DD4BF → #0D9488
- 背景：#FFFFFF（白色）
- 侧边栏：#102136（深蓝黑）
- 文字主色：#0F172A
- 文字辅色：#64748B
- 成功色：#0D9488
- 错误色：#DC2626

## 字体
- 系统默认：`system-ui, -apple-system, sans-serif`
- 正文无特殊字体要求

## 登录页
- 窗口尺寸：420×640
- 布局：竖向单列
- 顶部：绿色渐变横幅（100px高），品牌名+版本号
- 中间：头像环（白色圆环+绿色渐变头像背景）
- 表单：账号输入框、密码框、选项行、登录按钮
- 底部：注册账号 | 卡密充值 | 解绑设备

## 主界面
- 窗口尺寸：1440×900，最小 1200×760
- 左侧：240px 深蓝色侧边栏（可折叠到64px）
- 顶部：白色内容头部（64px）
- 右侧：灰色背景内容区

## UI 组件规范
| 组件 | 样式 |
|------|------|
| 输入框 | 四边框 + 圆角10px + 浅灰背景#F8FAFB + 高度46px |
| 按钮 | 胶囊形圆角23px + 绿色渐变 + 柔和阴影 |
| 密码框 | PrimeVue Password组件 + 眼睛图标切换 |
| 复选框 | PrimeVue Checkbox + 绿色勾选 |
| 下拉选择 | 白色浮层 + 圆角10px + 柔和阴影 |
| 弹窗 | 白色背景 + 圆角16px + 大阴影 |
| 卡片 | 白色 + 圆角18px + 柔和阴影 box-shadow: 0 10px 30px rgba(15,23,42,0.06) |
| 时钟 | 白色大字2.2rem + 普通字体 |
| 提示消息 | 成功：绿底绿字 / 错误：红底红字 |
| 链接 | 绿色文字 #0D9488，hover 变深绿 |
| 分隔线 | 浅灰 1px 实线 #F1F5F9 |
| 头像框 | 绿色渐变圆环，白色外圈 |
| 菜单高亮 | 绿色半透明渐变背景 |
